Lesson 25 : Computercode

Computercode elements

  • <kbd> for keyboard input

    The html <kbd> element is used to define keyboard input. The content inside is displayed in the browser's default monospace font

    Save the document by pressing Ctrl + S

    ex:
    <p>Save the document by pressing <kbd>Ctrl + S</kbd><p>

  • <samp> for program output

    The html <samp> element is used to define sample output. The content inside i displayed in the browser's default monospace font.

    Message from my computer:

    File not found.
    Press F1 to continue

    ex:
    <p>Message from my computer:</p>
    <p><samp> File not found. <br> Press F1 to continue </samp></p>

  • <code> for computer code

    <code> element is used to define a piece of computer code.

    x = 5; y = 6;s z = x + y;
    ex: 
            <code>
            x = 5;
            y = 6;
            z = x + y;
        </code>

  • <var> for variables

    The html <var> element is used to define a variable in programming or in a mathematical expression

    The area of a triangle is: 1/2 x b x h, where b is the base, and h is the vertical height.

    ex:
    <p>The area of a triangle is: 1/2 x <var>b</var> x <var>h</var>, where <var>b</var> is the base, and <var>h</var> is the vertical height. </p>
