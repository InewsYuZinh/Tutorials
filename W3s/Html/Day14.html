Lesson 24 : Responsive

Setting The Viewport

To create a responsive website, add the following <meta> tag to all your web pages:

ex:
< meta name="viewport" content="width=device-width, initial-scale=1.0" >

Responsive Images

  • Using the width property

    If the CSS width property is set to 100%, the image will be responsive and scale up and down:

    ex:
    < img src="picture.jpg" style="width:100%;" >
    notice:

    the image can be scaled up to be larger than its original size. A better solution, in many case, will be to the max-width property instead

  • Using the max-width property

    If the max-width property is set to 100%, the image will scale down if it has to, but never scale tup to be large than its original size:

    ex:
    < img src="picture.jpg" style="max-width: 100%; height :auto;" >

  • Show diffence images depending on browser width

    The html <picture> element allows you to define different images for different browser window sizes

    ex: 
        <picture>
        <source srcset:"img_smallflower.jpg media="(max-width: 600px)">
        <source srcset:"img_flowers.jpg media="(max-width: 1500px)">
        <source srcset:"flowers.jpg>
        <img src="img_smallflower" alt="Flowers">
    </picture>
