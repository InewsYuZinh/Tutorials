Lesson 21 : File Paths

File Paths Example

< img src="picture.jpg">

file is located in the same folder as the current page

< img src="images/picture.jpg">

file is located in the images folder in the current folder

< img src="/images/picture.jpg">

file is located in the images folder at the root of the current web

< img src="../picture.jpg">

file is located in the folder one level up from the current folder

Lesson 23: Layout

HTLM Layout Element

  • < header> - Defines a header for a document or a section
  • < nav> - Defines a set of navigation links
  • < section> - Defines a section in a document
  • < article> - Define an independent, self-contained content
  • < aside> - Define content aside from the content (like a sidebar)
  • < footer> - Defines a footer for a document or a section
  • < details> - Defines additional details that the user can open and close on demand
  • < summary> - Define a heading for the < details> element
